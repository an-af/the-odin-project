CSS Clamp, Min & Max

Praktek langsung untuk memahami fungsi CSS modern ini

ðŸŽ¯ Clamp()

Fungsi clamp() menggabungkan min() dan max() dalam satu fungsi. Formatnya: clamp(min, preferred, max)

Contoh Penggunaan

font-size: clamp(1rem, 4vw, 2.5rem);

Artinya: ukuran font minimal 1rem, ideal 4vw, maksimal 2.5rem

TEXT INI AKAN BERUBAH UKURAN

Contoh Lain: Lebar Kontainer

width: clamp(300px, 80%, 800px);

Kontainer lebarnya minimal 300px, ideal 80% dari parent, maksimal 800px

ðŸ”½ Min()

Fungsi min() memilih nilai terkecil dari beberapa nilai yang diberikan.

Contoh Penggunaan

width: min(50%, 600px);

Artinya: lebar akan mengambil nilai terkecil antara 50% atau 600px

KOTAK INI

Contoh Lain: Font Size

font-size: min(2rem, 10vw);

Ukuran font akan mengambil yang lebih kecil antara 2rem atau 10vw

ðŸ”¼ Max()

Fungsi max() memilih nilai terbesar dari beberapa nilai yang diberikan.

Contoh Penggunaan

width: max(200px, 30%);

Artinya: lebar akan mengambil nilai terbesar antara 200px atau 30%

KOTAK BESAR

Contoh Lain: Margin

margin: max(20px, 5vw);

Margin akan mengambil yang lebih besar antara 20px atau 5vw
