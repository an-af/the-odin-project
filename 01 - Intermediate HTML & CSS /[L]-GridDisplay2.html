Tantangan #1: Layout 3x3 Grid

Buat sebuah layout grid dengan 9 kotak yang disusun dalam 3 baris dan 3 kolom. Setiap kotak harus memiliki ukuran yang sama dan memiliki jarak yang konsisten antar kotak.

Petunjuk (Jangan Dibuka Dulu!)

  • Gunakan display: grid pada container utama
  • Atur grid-template-columns untuk membuat 3 kolom
  • Gunakan gap untuk memberi jarak antar kotak
  • Setiap kolom bisa menggunakan 1fr untuk ukuran yang sama
  • Jangan lupa atur height pada kotak agar terlihat

âœ… Solusi:

.container { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 15px; } .grid-item { height: 80px; background: #3498db; /* Warna bisa diganti */ border-radius: 8px; display: flex; align-items: center; justify-content: center; color: white; font-weight: bold; }

Penjelasan:

grid-template-columns: 1fr 1fr 1fr membuat 3 kolom dengan ukuran yang sama. fr (fraction) berarti setiap kolom mendapatkan bagian yang sama dari ruang yang tersedia.

gap: 15px memberikan jarak 15px antara semua kotak, lebih baik daripada menggunakan margin karena tidak membuat masalah dengan alignment.

Dengan hanya 3 properti CSS ini, kamu sudah bisa membuat layout grid yang sempurna! Ini adalah 20% fitur Grid yang memberikan 80% hasil.

Tantangan #2: Card Layout Responsif

Buat layout card yang otomatis menyesuaikan jumlah kolom berdasarkan ukuran layar. Di layar besar: 3 kolom, di tablet: 2 kolom, di mobile: 1 kolom.

ðŸŽ¯ Hasil yang Diharapkan:

Layout card yang responsif tanpa media query!

Petunjuk (Jangan Dibuka Dulu!)

  • Gunakan repeat(auto-fit, ...) untuk membuat kolom yang fleksibel
  • Gunakan minmax(250px, 1fr) untuk ukuran kolom
  • auto-fit membuat grid menyesuaikan jumlah kolom secara otomatis
  • minmax() memastikan kolom tidak terlalu kecil atau terlalu besar
  • Tambahkan gap untuk jarak yang konsisten

âœ… Solusi:

.container { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 20px; } .card { height: 120px; background: #3498db; border-radius: 10px; display: flex; align-items: center; justify-content: center; color: white; font-weight: bold; }

Penjelasan:

repeat(auto-fit, minmax(250px, 1fr)) adalah kombinasi ajaib untuk layout responsif! Ini berarti:

â€¢ auto-fit: Grid akan membuat sebanyak mungkin kolom yang muat

â€¢ minmax(250px, 1fr): Setiap kolom minimal 250px, maksimal 1fr (satu bagian dari ruang tersedia)

Dengan satu baris code ini, layout akan otomatis menjadi 1 kolom di mobile, 2 di tablet, dan 3+ di desktop. Ini adalah fitur Grid paling powerful!
