Contoh Praktis (Pareto 80/20)

1 Layout Blog/Grid Card

Grid: Perfect untuk card layout dengan kolom yang responsif. Cukup 1 baris code!

grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));

2 Header/Navbar

Flexbox: Ideal untuk menyusun item dalam satu baris dengan space-between.

display: flex; justify-content: space-between; align-items: center;

3 Form Layout

Grid: Bisa atur label dan input berdampingan dengan mudah di semua ukuran layar.

display: grid; grid-template-columns: 1fr 2fr; gap: 10px;

4 Centering

Flexbox: Masih raja untuk centering vertikal & horizontal dalam satu dimensi.

display: flex; justify-content: center; align-items: center;

Cheat Sheet: Kapan Pakai Grid vs Flexbox?

Pakai Grid Ketika...

  • Buat layout halaman (header, sidebar, content, footer)
  • Butuh layout dua dimensi (baris & kolom)
  • Ingin membuat card layout responsif
  • Perlu mengatur grid area (named areas)
  • Buat masonry layout atau layout kompleks

Pakai Flexbox Ketika...

  • Menyusun item dalam satu baris/kolom
  • Butuh centering vertikal & horizontal
  • Membuat navbar atau toolbar
  • Distribusi ruang antar elemen
  • Responsive dalam satu dimensi

Pro Tips!

  • Grid = "parent" mindset, Flexbox = "child" mindset
  • Grid untuk layout besar, Flexbox untuk komponen kecil
  • Bisa digabung! Grid di luar, Flexbox di dalam
  • minmax() + auto-fit = responsif tanpa media query
  • gap lebih baik dari margin untuk spacing
