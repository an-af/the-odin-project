CSS Position, Penting ?

Ga Ada position = Bangun rumah tanpa pondasi

1, Position : static

Elemen mengikuti flow normal, Tidak bisa pake top/bottom/left/right

Ini tetap ga bisa di pindah ya.. :-)

2. position relative;

Bisa dipindah dari posisi aslinya, tapi tetep di flow dokumen

Aku relative! -> Geser 20px

3, Position absolute;

Lepas dari flow, mengikuti parent yang berposisi (relatif / absolut)

parent aktif

Aku terbang !

4, Position fixed;

Tetap di layar meskipun di scroll, Cocok buat navbar atau button penting

Scroll terus, lihat kanan bawah

Aku Tetap disini

5, Position: sticky;

Campuran relative + fixed "Nempel" Saat mencapai posisi tertentu

Scroll ke bawah ->

Aku nemple saat mencapai top:80px

Ini hanya scroll test, Sticky bekerja

Kenapa Positioning Penting?

  • ✅ Bikin navbar, modal, tooltip
  • ✅ Kontrol layout secara presisi
  • ✅ Buat UI modern (card, popup, fixed button)
  • ❌ Tanpa positioning → desain kaku & broken

/* Contoh penting */


.modal {

position: fixed;

top: 0; left: 0;

width: 100%; height: 100%;

background: rgba(0,0,0,0.5);

display: flex;

justify-content: center;

align-items: center;


}
