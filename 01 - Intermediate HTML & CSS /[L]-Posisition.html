Why Positioning Matters

Belajar CSS tanpa positioning = bikin rumah tanpa fondasi

1. position: static;

Elemen mengikuti flow normal. Tidak bisa pakai top/left/right/bottom.

Ini tetap di flow-nya. Gak bisa dipindah!

2. position: relative;

Bisa dipindah dari posisi aslinya, tapi tetap di flow dokumen.

Aku relatif! ‚Üí Geser 20px

3. position: absolute;

Lepas dari flow, mengikuti parent yang berposition (relative/absolute).

Parent relatif
Aku terbang!

4. position: fixed;

Tetap di layar meskipun discroll. Cocok buat navbar atau button penting.

Aku tetap di sini!

5. position: sticky;

Campuran relative + fixed. "Nempel" saat mencapai posisi tertentu.

Ini hanya scroll test. Sticky bekerja!
